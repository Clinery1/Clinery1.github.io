▸ About Me

I am a freshman in Florida Polytechnic University pursuing a degree in Computer Science. I have been programming since 2016, but I started with websites. I started learning actual programming languages about a year after that. The first programming language I learned was Rust. I am very glad I learned Rust first because it allowed me to learn the theory parts of programming without the headaches that other languages like C impose (looking at you pointers and array-out-of-bounds errors). I eventually learned more languages (see Languages I know) and started programming games.
